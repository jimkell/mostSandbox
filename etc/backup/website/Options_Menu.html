Set Up Solver:

Click Options -> Set Up Solver to select beween GLPK and Gurobi solvers. Gurobi selection is not enabled if Gurobi 5.6.2 or better is not installed. If Gurobi is already installed, and the version of Gurobi is 5.6.2 or better the dialog will display the following message:

Otherwise the following message will be displayed: (The message may say 32 or 64 bit depending on the version of Java installed.)

If Gurobi is installed and the version is 5.6.2 or better, just click the Gurobi button, then OK, and Gurobi is set up as a solver. If a license for Gurobi is not installed, a license can be downloaded at the Gurobi site www.gurobi.com. Otherwise follow the Gurobi installation instructions here.
